# Athlete Injury Risk Detection — live demo

Live dashboard for [athlete-injury-risk-detection](https://github.com/Amayyas/athlete-injury-risk-detection).

Predicts the calibrated probability that an athlete suffers a muscle injury within the
next 7 days, alongside a transparent rule-based score and a live SHAP explanation.

This Space runs the project's published container image
(`ghcr.io/amayyas/athlete-injury-risk-detection`), which ships a trained model — so the
model predictions are live, not a placeholder.
